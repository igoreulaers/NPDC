# Norwegian Polar Data Centre

Here I provide code to access the following NPDC databases:
* field
* ecotox

There is a library of queries created, each delivering a dataframes including contaminant and metadata. 

Each query comes in four different formats:
* simple: quick 'n' dirty with essential data (species, matrix, locality, data)
* extended field: essential data are extended to include all field metadata
* extended lab: essential data are extended to include all lab metadata
* extended field and lab: essential data are extended to include all field and lab metadata

Queries are available for the following projects:
* MOSJ
* COPOL
