# Norwegian Polar Data Centre

Here I provide a library of search queries, each delivering a specific pollutant dataset (including metatda), of the Norwegian Polar Institute Data Centre.

Each query offers four different output formats:
* simple: quick 'n' dirty with essential data (species, matrix, locality, data)
* extended field: essential data are extended with field metadata
* extended lab: essential data are extended with metadata
* extended field and lab: essential data are extended with field and lab metadata

Queries are available for the following projects:
* MOSJ
* COPOL

Queries are available for the following species:
* polar bear
* glaucous gull
* black-legged kittiwake
* common eider
